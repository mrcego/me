# César Gómez | Senior Frontend Architect & Fullstack Engineer

A high-fidelity, cinematic portfolio built with **Nuxt 3**, **Vue 3**, and **Motion-V**. Designed with a "Megawow" aesthetic, featuring glassmorphism, advanced micro-interactions, and full internationalization.

## ✨ Key Features

- **Cinematic UX/UI**: Premium dark-mode aesthetic with glassmorphism and holographic HUD elements.
- **Full Internationalization (i18n)**: One-click toggle between English and Spanish with persistent locale detection.
- **Dynamic Theming**: Interactive color picker to customize primary accents and surface tones in real-time.
- **Performance Optimized**:
  - SSR (Server-Side Rendering) for SEO.
  - Client-side hydration fixes for flicker-free transitions.
  - Optimized image delivery via `@nuxt/image`.
- **Responsive Mastery**: Tailored experiences for mobile, tablet, and desktop with fluid typography.

## 🛠️ Tech Stack

- **Framework**: [Nuxt 4](https://nuxt.com/) (Vue 3)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) con Vite
- **Animations**: [Motion-V](https://motion-v.com/) (Motion for Vue)
- **Icons**: [Lucide Vue Next](https://lucide.dev/) + [Nuxt Icon](https://nuxt.com/modules/icon) (Solar, Simple Icons, Logos)
- **Internationalization**: [@nuxtjs/i18n](https://i18n.nuxtjs.org/)
- **UI Components**: [PrimeVue 4](https://primevue.org/) con tema Aura
- **SEO**: [@nuxtjs/seo](https://nuxtseo.com/) + Robots + Sitemap
- **Image Optimization**: [@nuxt/image](https://nuxt.com/modules/image)
- **Fonts**: [@nuxt/fonts](https://nuxt.com/modules/fonts)
- **Utilities**: [VueUse](https://vueuse.org/)
- **Development**: ESLint + Nuxt Hints

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+)
- npm / pnpm / yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mrcego/me-v2.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 📦 Build & Deployment

Build the production application:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
